# Parallax Protocol Foundry VTT prototype

This is a prototype V14 system folder for Parallax Protocol focused on:

- a custom `character` actor type
- Application V2 / DocumentSheetV2 + Handlebars sheet UI
- item types for weapons, armor, fields, gear, and species traits
- derived calculations for skills, saves, initiative, load, movement, and DR
- basic roll buttons for skills, saves, initiative, and weapon attacks

## Current scope

This is a first-pass playable sheet prototype. It is not a complete rules engine yet.

## Known rough edges

- Only the `character` actor type has a custom sheet right now.
- NPC / robot / creature actor types are declared but not yet implemented.
- Opposed targeting automation is not implemented yet.
- Some Foundry V14 specifics may need small adjustments once tested in a live world.
